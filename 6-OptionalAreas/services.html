About this page: Here, we added navigation with items that can be market Active on each individual content page to highlight the current active menu item. In this project index.html is the master page and the front page at the same time.

Try:

  1. Take a look at how each menu Nav item is defined as Editable area. By default, editable areas have editable inner content. But here, we disable editing the content and insted define which CSS classes can be added or removed from the element.
  2. Edit menu labels and / or links on the master page. You can even reorder the menu items. Do "Components -> Update" to apply changes to all content pages. The menu items will still work and be highlighted correctly.
  3. Add a class other than "active" on any menu item and update. Because this class is not listed as editable it is applied to all instances of this menu item across content pages.
  4. The master page is also our front index page. That's convenient for smaller projects like this. No need to keep a separate master page.
