About this page: Here, we took the navigation example and made page head information editable: title, page description and page keywords, so that every page can have different settings.

Try:

  1. Open index.html (the master page), uncollapse the head element in the tree and check out how title, meta description and meta keywords are defined as editable areas with actions in ACT panel.
  2. Open any of the child pages and notice that every child page has different values in these fields. These values are preserved if you do Components -> Update.  
  3. You can edit page head properties in PROP panel when the top page element is selected in the tree.
  4. At the moment it is not possible to make only one part of title or keywords editable (so that you could define some general keywords on the master page and specific ones on child pages). That means that you have to manually edit information on child pages, even if you change values on the master page.
